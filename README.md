# Тестовое задание

Проект создан по ТЗ: верстка блока с текстом и изображениями, добавлен адаптив.

## Особенности проекта
- HTML + CSS (БЭМ)
- Используются только CSS без JavaScript
- Кнопка оформлена средствами CSS
- Все изображения можно перемещать, добавлять или удалять, плюс можно регулировать по ширине, менять/создавать новые


## Просмотр проекта
- Локально: открыть `index.html` в браузере
---

### Инструкция
1. Скачать репозиторий
2. Открыть файл `index.html` в любом браузере
